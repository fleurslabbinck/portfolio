About

For this project, I had to extend upon a custom engine from a gameplay programming course at DAE. I implemented things learned in that course like steering behaviors, pathfinding, behavior trees, finite state machines ... The purpose of this project is very straightforward: survive in a zombie environment.

What I Learned

This project was really eye opening, especially the game AI part. For the research part of the course, I studied a hybrid between behavior trees and finite state machines.
