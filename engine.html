About

This project, combined with the Bomberman remake was part of an exam assignment for a DAE course. It is a 2D game engine with an SDL 2 framework for arcade style games, written in purely C++ according to the C++ Core Guidelines. The engine builds further upon the Minigin start project, which is nowhere near a game engine yet and only contains the bare minimum. The book Game Programming Patterns by Robert Nystrom provided great guidence during the process of implementing all kinds of game patterns. The engine was successfully put to the test by the recreation of Bomberman.
