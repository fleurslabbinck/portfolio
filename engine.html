2D Engine

Project

During the last programming course, I created this 2D engine. The engine built further upon the Minigin start project, which is nowhere near a game engine yet and only contains the bare minimum. The book Game Programming Patterns by Robert Nystrom provided great guidence during the process of implementing all kinds of game patterns like Game Object and Component, Game Loop, Update Method, Dirty Flag, Command, Observer, Event Queue, Service Locator, Singleton (although cautiously) and State. The engine was successfully put to the test by the recreation of Bomberman.

The whole project is written in C++ and follows the C++ Core Guidelines.

What I learned

This was by far my most interesting project yet. Getting to know how game engines work was invaluable and improved me as a gameplay programmer in general. I have learned so much during the whole process and I can't wait to keep learning.
